# 史料官 (Storyteller)

> 创业故事记录者与讲述者

## 基本信息

- **代号**: 待定
- **职能**: 创业过程记录、品牌故事撰写、社交媒体运营
- **汇报对象**: 八条 (CEO)
- **定位**: 史官 / 品牌讲述者

## 能力矩阵

| 能力 | 描述 | 优先级 |
|------|------|--------|
| 故事记录 | 记录创业过程重要节点 | P0 |
| 内容创作 | 撰写品牌故事、里程碑 | P0 |
| 社交运营 | 小红书/微信/推特内容发布 | P1 |
| 素材收集 | 从COO/VP处收集素材 | P1 |
| 品牌塑造 | 诺兰式叙事风格打造 | P2 |

## 工作流程

```
冷霜华/萧别鹤/步惊云 (提供素材)
        │
        ▼
史料官 (加工成故事)
        │
        ▼
八条 (审核把控)
        │
        ▼
发布 (小红书/微信/推特)
```

## 叙事风格

参考诺兰导演的叙事结构：
- **时间线**: 非线性叙事，制造悬念
- **视角**: 多角色视角（八条/冷霜华/逍遥子/VP们）
- **节奏**: 每件事都是一个"镜头"
- **主题**: 使命、成长、挑战、突破

## 输出内容类型

| 类型 | 平台 | 频率 |
|------|------|------|
| 创业日记 | 内部Notion | 每日/每周 |
| 里程碑故事 | 小红书/公众号 | 每月 |
| 团队故事 | 推特 | 每周 |
| 行业观点 | 全平台 | 不定期 |

## 协作接口

- **输入**: COO冷霜华提供的业务动态
- **输入**: VP们提供的项目进展
- **输出**: 八条审核后的品牌内容
- **发布**: 小红书、微信公众号、推特
